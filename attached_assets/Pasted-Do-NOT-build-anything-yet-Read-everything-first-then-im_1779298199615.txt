SECTION 1 — FIX TASK DETECTION (Root cause of all AI problems)
===================================================================

The core problem: messages like "حدث عقلك", "نفذ", "اعمل حملة كاتلوج" 
are being classified as "chat" mode instead of "execute".
And messages like "الميزانية اليومية كام" are classified as "chat" instead of "analyze".

In detectTaskMode(), replace the entire function with this:

```typescript
function detectTaskMode(messages: { role: string; content: string }[]): TaskMode {
  const last = messages.filter((m) => m.role === "user").at(-1)?.content ?? "";
  const t = last.toLowerCase();

  // Execute — Arabic + English
  if (/\b(create|build|launch|make|set up|setup|execute|duplicate|scale up|pause all|enable all|deploy|نفذ|اعمل|انشئ|شغل|فعل|ابدا|ابدأ|عدل|غير|زود|قلل|نسخ|اعمله|اطلقه|وقف الحمله|وقف الحملة|شغل الحمله|شغل الحملة|ارفع الميزانيه|خفض الميزانيه|حملة كاتلوج|حمله كاتلوج|حملة جديده|حمله جديده)\b/.test(t)) return "execute";

  // Brain update — always execute mode
  if (/\b(حدث عقلك|حدث العقل|احفظ|save brain|update brain|حفظ البيانات)\b/.test(t)) return "execute";

  // Analyze — Arabic + English  
  if (/\b(audit|analyze|analyse|check|review|report|show|tell|what is|what's|how is|how are|performance|stats|breakdown|trend|compare|explain|why|roas|ctr|cpm|spend|صرفت|جابت|شوفلي|كام|اليوم|انهردا|الحمله|الحملة|بيانات|أداء|ادا|نتايج|نتائج|تقرير|إيه|ايه|عامله|عامل|شغاله|شغال|كيف|اتوقف|فين|مين|امتى|الميزانيه|الميزانية|البادجت|budget|يومي|daily|حلل|تحليل|سيجمنت|جمهور|اوديانس|الحسابات|الحملات|المجموعات)\b/.test(t)) return "analyze";

  // Plan
  if (/\b(plan|strategy|recommend|suggest|structure|approach|best way|how should|what should|advise|idea|next step|خطه|خطة|استراتيجيه|استراتيجية|نصيحه|نصيحة|ايه الافضل|ايه احسن|اقترح|افضل طريقه|باير بيرسونا|buyer persona|سيجمنتات|segments)\b/.test(t)) return "plan";

  return "chat";
}
```

In the Reports component, replace the date range selector with:

```typescript
const DATE_PRESETS = [
  { label: "اليوم", value: "today", days: 0 },
  { label: "أمس", value: "yesterday", days: 1 },
  { label: "آخر 7 أيام", value: "last_7", days: 7 },
  { label: "آخر 30 يوم", value: "last_30", days: 30 },
  { label: "آخر 90 يوم", value: "last_90", days: 90 },
  { label: "آخر 365 يوم", value: "last_365", days: 365 },
  { label: "هذا الشهر", value: "this_month", days: -1 },
  { label: "هذه السنة", value: "this_year", days: -2 },
  { label: "مخصص", value: "custom", days: -3 },
];

function getDateRange(preset: string, customFrom?: string, customTo?: string) {
  const today = new Date();
  const format = (d: Date) => d.toISOString().split("T")[0];
  
  switch(preset) {
    case "today":
      return { since: format(today), until: format(today) };
    case "yesterday":
      const yesterday = new Date(today);
      yesterday.setDate(today.getDate() - 1);
      return { since: format(yesterday), until: format(yesterday) };
    case "this_month":
      return { since: format(new Date(today.getFullYear(), today.getMonth(), 1)), until: format(today) };
    case "this_year":
      return { since: `${today.getFullYear()}-01-01`, until: format(today) };
    case "custom":
      return { since: customFrom!, until: customTo! };
    default:
      const days = DATE_PRESETS.find(p => p.value === preset)?.days ?? 30;
      const from = new Date(today);
      from.setDate(today.getDate() - days);
      return { since: format(from), until: format(today) };
  }
}
```

Show custom date pickers (from/to) only when "مخصص" is selected.
Make sure the selected date range is passed correctly to ALL API calls in the reports page.
