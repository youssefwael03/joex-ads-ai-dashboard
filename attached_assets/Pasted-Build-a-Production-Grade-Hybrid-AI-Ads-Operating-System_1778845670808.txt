## MAIN ARCHITECTURE GOAL

Convert the assistant from:
"dynamic AI tool caller"

Into:
"Hybrid AI Decision + Deterministic Execution Engine"

The AI should:

* THINK strategically
* DECIDE intelligently
* EXECUTE through backend templates and services

NOT generate huge payloads dynamically every time.

---

Folder:
backend/templates/

1. Catalog Sales Template
2. Broad Scaling Template
3. Retargeting Template
4. Lead Generation Template
5. WhatsApp Campaign Template
6. Advantage+ Template
7. ABO Testing Template

Templates must be executable without AI-generated payloads.

2. AI = DECISION ENGINE ONLY

---

The AI should NOT:

* manually construct massive campaign payloads
* dynamically generate every API field
* repeatedly fetch Meta structures

Instead:
AI returns compact strategic decisions.

Example:

{
"template": "catalog_scaling",
"budget": 600,
"targetCountries": ["EG"],
"bestAudience": "Women 25-44",
"creativeStyle": "UGC",
"optimizationGoal": "Purchase"
}

Then backend executors build the actual Meta payloads.
