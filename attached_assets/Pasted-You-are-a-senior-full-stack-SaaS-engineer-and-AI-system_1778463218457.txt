You are a senior full-stack SaaS engineer and AI systems architect.

Build a production-grade AI-powered advertising analytics platform called “Joex Ads Dashboard”.

This is NOT a simple reporting dashboard.

The platform should function as a complete Performance Marketing Operating System for media buyers, agencies, ecommerce brands, and performance marketers.

The system must deeply integrate with Meta Marketing API using a user-provided long-lived access token (NO OAuth login).

The user pastes their own token → selects an ad account → the entire platform updates dynamically based on that account.
